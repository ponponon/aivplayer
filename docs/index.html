AIVPlayer

本地 AI 字幕生成的跨平台视频播放器

基于 whisper.cpp 的离线语音识别，无需联网即可为视频自动生成高质量字幕。 你的视频和字幕数据始终留在自己手里。

GitHub 下载 Gitee 下载（国内）

支持 macOS · Windows · Linux

功能特性

强大的本地 AI 能力，保护你的隐私

本地 AI 字幕

基于 whisper.cpp 的离线语音识别，无需联网即可为视频自动生成高质量字幕，支持中英日韩等多语言。

多格式支持

支持 MP4、WebM、MOV、MKV、AVI 等主流视频格式，拖拽导入或文件选择器打开。

片段导出

基于当前播放位置快速导出短片段，支持 15/30/60 秒三档长度，可选纯视频、外挂字幕或字幕烧录。

字幕翻译

支持将生成的字幕翻译为目标语言，使用 OpenAI 兼容接口，支持自定义术语表。

截图录屏

截取当前画面为图片，录制指定时长的视频片段，支持 GIF 导出。

多语言界面

支持简体中文、English、日本語、한국어 四种语言，系统语言自动匹配。

界面预览

现代化的深色影院风格，视频优先

快速开始

从源码构建或直接下载使用

1 直接下载（推荐）

从 Release 页面下载对应平台的安装包，双击即可安装使用。

下载地址：

GitHub Releases Gitee Releases（国内）

2 从源码构建

适合开发者，需要 Node.js 18+ 环境。

# 克隆仓库
git clone https://github.com/ponponon/aivplayer.git
cd aivplayer

# 安装依赖
npm install

# 启动开发模式
npm run dev

# 构建生产版本
npm run build

# 打包安装程序
npm run dist
