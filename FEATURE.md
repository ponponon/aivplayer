# AIVPlayer 功能列表

## 核心播放
- 本地视频播放器，支持拖拽导入和文件选择导入。
- 播放列表支持切换上一条 / 下一条，支持键盘快捷键。
- 支持音量、倍速、全屏和时间轴拖动控制。

## 本地 ASR 字幕
- 基于 whisper.cpp 的离线字幕生成流程。
- 支持模型下载、运行时检测、自动检测 whisper 引擎、手动选择引擎。
- 支持字幕缓存，重复生成同一视频时优先命中本地结果。
- 生成字幕时同时输出 VTT 和 SRT 两份文件，VTT 直接挂载到播放器，SRT 便于导出和外部使用。
- 重新打开同一个视频时会自动探测字幕缓存，命中后直接回填字幕轨道和状态，不用重新生成。
- 生成字幕后可一键打开字幕缓存文件夹，或直接定位 SRT 文件，方便检查 VTT / SRT 文件和调试挂载结果。
- 生成字幕后可以直接用系统默认应用打开 SRT 文件，不再把“打开文件”误做成“在文件夹里定位”。
- 字幕卡片会显示简短状态标签，直接告诉用户当前是“等待生成 / 缓存已就绪 / 已挂载”，减少猜测成本。
- 如果手头只有 VTT，也可以从当前字幕一键重新导出一份 SRT，继续给外部工具使用。
- 字幕操作菜单还可以直接复制 VTT / SRT 路径，方便丢给终端脚本、批处理或外部工具。
- 字幕操作菜单支持点击外部或按 `Esc` 自动关闭，避免菜单停留在半开状态。
- 信息 tab 会直接展示当前媒体文件、播放状态和字幕缓存状态，不再只显示一个空壳提示。
- 信息 tab 里播放状态和字幕缓存用 compact 双列卡片展示，把短信息收紧，减少空白和错位感。
- 导出和转换字幕时会保留 cue 内的换行结构，避免多行字幕被压成单行。
- 导出 SRT 时会自动清理 VTT 的 `voice / i / b / u / c` 等常见内联标签，减少外部工具兼容问题。
- 导出 SRT 时会去掉 `ruby / rt / rp` 的注音标记，保留正文文字。
- 导出 SRT 时会解码常见 HTML 实体，例如 `&amp; / &lt; / &gt; / &nbsp;`。
- 导出 SRT 时会解码更多常见排版实体，例如省略号、长破折号和引号。
- 导出 SRT 时会容忍非法数值实体，避免脏字幕文件直接导致转换失败。
- 生成 VTT 时会自动转义普通文本里的 `& / < / >`，确保输出始终是合法 VTT。

## 模型下载
- 推荐模型支持国内 ModelScope 和国际 Hugging Face 两个来源。
- 下载进度在侧边栏展示，方便定位卡顿或失败。

## ASR 二进制兼容
- 自动识别 `whisper-whisper-cli`、`whisper-cli`、`whisper-cpp`、`main` 这几类 whisper.cpp 可执行文件。
- 如果用户选到的是 deprecated wrapper，运行时会优先切换到同目录下真正可用的二进制。
- 资源打包时尽量保留识别到的原始二进制文件名，避免把新版本 whisper.cpp 再改回旧名字。
